CuriousBooks API Reference

Base URL: http://localhost:5000/api

Endpoints

  • 🔐 Authentication
  • ⭐ Reviews

  • 💡 Recommendations

📚 Books API

Endpoints for listing, searching, and retrieving book information.

GET /books

Get all books with optional filtering and pagination.

Query Parameters

NameTypeDescription
pageintPage number (default: 1)
per_pageintItems per page, max 100 (default: 20)
category_idintFilter by category ID
genrestringFilter by genre name
min_pricefloatMinimum price filter
max_pricefloatMaximum price filter
in_stockboolOnly return books with stock > 0
sort_bystringColumn to sort by (default: popularity_score)
sort_orderstring'asc' or 'desc' (default: desc)

Response

{ "books": [{ "id": 1, "title": "...", ... }], "total": 150, "pages": 8, "currentPage": 1, "hasNext": true, "hasPrev": false }
200 OK
GET /books/featured

Get featured/popular books for homepage display.

Query Parameters

NameTypeDescription
limitintNumber of books to return (default: 4)
200 OK
GET /books/search

Search books by title, author, description, genre, or ISBN.

Query Parameters

NameTypeDescription
qstringSearch query required
pageintPage number (default: 1)
per_pageintItems per page (default: 20)
200 OK
GET /books/{id}

Get detailed information for a single book by ID.

Path Parameters

NameTypeDescription
idintBook ID required
200 OK 404 Not Found
GET /books/{id}/reviews

Get all reviews for a specific book with pagination.

200 OK 404 Not Found
GET /books/genres

Get list of all unique genres in the database.

200 OK

🔐 Authentication API

User registration, login, and account management. Uses JWT tokens.

POST /auth/register

Register a new user account.

Request Body

{ "email": "user@example.com", // required "password": "securepass123", // required "username": "johndoe", // required "firstName": "John", // optional "lastName": "Doe" // optional }

Response

{ "message": "User registered successfully", "user": { ... }, "accessToken": "eyJhbGci...", "refreshToken": "eyJhbGci..." }
201 Created 400 Bad Request 409 Conflict
POST /auth/login

Authenticate user and receive JWT tokens.

Request Body

{ "email": "user@example.com", "password": "securepass123" }
200 OK 400 Bad Request 401 Unauthorized
POST /auth/refresh 🔒 Auth

Refresh expired access token using refresh token.

Send refresh token in Authorization header: Bearer {refreshToken}

200 OK 401 Unauthorized
GET /auth/me 🔒 Auth

Get current authenticated user's profile.

200 OK 401 Unauthorized
PUT /auth/me 🔒 Auth

Update current user's profile information.

200 OK 401 Unauthorized 409 Conflict

⭐ Reviews API

Create, read, update, and delete book reviews. One review per user per book.

GET /reviews/book/{bookId}

Get all reviews for a specific book with average rating.

200 OK
POST /reviews 🔒 Auth

Create a new review for a book.

Request Body

{ "bookId": 123, // required "rating": 4.5, // required, 0-5 "text": "Great book!" // optional }
201 Created 400 Bad Request 409 Conflict
PUT /reviews/{id} 🔒 Auth

Update an existing review (owner only).

200 OK 403 Forbidden
DELETE /reviews/{id} 🔒 Auth

Delete a review (owner only).

200 OK 403 Forbidden

🛒 Orders API

Shopping cart validation and order management. Handles checkout flow.

POST /orders/cart/validate

Validate cart items - check stock and calculate totals. No auth required.

Request Body

{ "items": [ { "bookId": "123", "quantity": 2 }, { "bookId": "456", "quantity": 1 } ] }

Response

{ "valid": true, "items": [{ "bookId": ..., "unitPrice": ..., "subtotal": ..., "available": ... }], "subtotal": 45.98, "tax": 3.68, "total": 49.66 }
200 OK
POST /orders/checkout 🔒 Auth

Create order from cart items. Validates stock, creates order, reduces inventory.

201 Created 400 Bad Request 404 Not Found
GET /orders 🔒 Auth

Get order history for current user with pagination.

200 OK
POST /orders/{id}/pay 🔒 Auth

Mark order as paid (mock payment processing).

200 OK 400 Bad Request
POST /orders/{id}/cancel 🔒 Auth

Cancel a pending order and restore inventory.

200 OK 400 Bad Request
